Open Source NSB Network
=======================

There is a strong shared interest across European National Standardization Bodies (NSBs) in exploring how Open Source (OS) approaches can strengthen and modernize the standardization ecosystem.
While experiences and maturity levels differ across countries, NSBs agreed that open collaboration, interoperability, and alignment with digital sovereignty goals should guide the joint agenda.
